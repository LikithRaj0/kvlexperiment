Experiment Procedure

  1. Click on start simulation to start the circuit.
  2. Set the DC voltage (0-30V).
  3. Set the resistance value (1 ohms - 200 ohms).
  4. Click on add table to add the required resistance and voltage values to the table
  5. The voltmeters are connected in parallel with the resistors.
  6. Consider loop1 from the circuit diagram and observe the values of voltage across the resistors R1 and R3.
  7. Check if the equivalent voltage across the resistors R1 and R3 is equal to the supply voltage existing in loop1.
  8. Now consider loop2 and check for the equivalent voltage across the resistors R2 and R3.
  9. If the equivalent voltages in loop2 is equal to zero, and if it satisfies the condition mentioned in point 5., then the Kirchoff's voltage law is proved.
